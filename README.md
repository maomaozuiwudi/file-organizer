# 🗂️ 文件整理助手

> **工具猫系列** — 智能桌面文件整理助手，自动分类归档文件

## ✨ 功能简介

一款轻量级的 Windows 桌面文件管理工具，提供三大核心功能：

### 1. 📁 归位整理
按文件扩展名自动分类归档到对应子文件夹。例如 `.jpg` → `jpg/`，`.pdf` → `pdf/`，让凌乱的桌面瞬间整洁有序。

### 2. 🔍 快速搜索 (FTS5)
基于 SQLite FTS5 全文索引引擎，秒级搜索电脑上的文件。选择目录建立索引后，输入关键词即可实时检索，双击结果直接打开文件。

### 3. 📦 一键迁移
按扩展名规则批量迁移文件（支持通配符），保留目录结构，适合从源目录向目标目录的有筛选搬迁。

## 🚀 使用方法

```bash
# 直接运行
python main.py

# 或使用打包版本
# 双击 文件整理助手.exe
```

### 界面操作

| 功能 | 操作步骤 |
|------|---------|
| 归位整理 | 选择目标文件夹 → 点击「开始整理」 |
| 快速搜索 | 点击「索引目录」选择索引范围 → 输入关键词实时检索 |
| 一键迁移 | 设定源目录、目标目录、扩展名规则 → 点击「迁移」 |

## 🛠️ 技术栈

- **语言**: Python 3.x
- **界面**: Tkinter (标准库)
- **索引**: SQLite FTS5 全文搜索
- **打包**: PyInstaller
